**Word Race** — это образовательная P2P мультиплеерная игра, созданная на движке Godot 4. Приложение превращает скучные школьные и университетские тесты в азартную 3D-гонку знаний, где контент генерируется нейросетями в реальном времени, а оценки выставляются ИИ за развернутые ответы, а не за угадывание чекбоксов.

Она создана для того, чтобы помочь учителям тратить 0 минут на подготовку тестов, а студентам — получать максимум удовольствия от проверки знаний и глубокую, понятную обратную связь по своим ошибкам.

## ✨ Ключевые особенности

- 🤖 **Бесконечный контент от ИИ**: Никаких заготовленных баз данных! Учитель вводит любую тему (от "Биологии Клетки" до "Истории Древнего Рима"), и ИИ (Llama-3 через Groq) на лету генерирует интересные, стимулирующие мышление открытые вопросы.
- 🗣️ **Проверка развернутых ответов**: Студенты отвечают своими словами. ИИ прощает опечатки, оценивает понимание сути ответа от 1 до 10 и оставляет **персонализированный комментарий** с разбором ошибки или похвалой. Защита от резервных сбоев реализована через запасной Google Gemini 2.5 API.
- 🎮 **3D-Геймификация и Соревнование**: Каждый правильный ответ увеличивает высоту колонны (пиллара), на которой стоит 3D-аватар студента. Это превращает тест в захватывающую "гонку в небо". Ошибки не наказывают, а лишь замедляют прогресс, давая шанс отыграться всем.
- 📶 **Абсолютно бесшовная P2P сеть**:
  - ✨ Подключение хоста (преподавателя) в 1 клик.
  - 🏠 В классе: автоматический поиск серверов внутри локальной сети (LAN Discovery).
  - 🌍 В интернете: генерация уникального 6-значного кода комнаты (поверх UPnP / Radmin VPN). Студенты просто вводят "2M7K9P" и они в игре! Никаких IP и портов.
- 📈 **Мощная встроенная аналитика для учителя**: Дашборд реального времени показывает средний балл класса, динамику успеваемости по дням, а также **список самых слабых тем** группы. Одним кликом можно выгрузить детальный отчет в `.txt` файл.
- 🎓 **Персональный План Изучения (Tutor AI)**: По завершении, студент может нажать кнопку "Сгенерировать План Изучения". ИИ проанализирует историю *всех* его ответов, найдет пробелы и сформирует пошаговый PDF/TXT-документ: что именно нужно повторить сегодня вечером и какие упражнения сделать, чтобы стать лучше.

## 🛠️ Архитектура проекта

Проект логически разделен на независимые модули-синглтоны (Autoloads), которые общаются через сигналы.

| Модуль | Описание |
| :--- | :--- |
| 🌐 `NetworkManager.gd` | Ядро мультиплеера. Работает по архитектуре "Хост-Клиент" (P2P). Управляет подключениями, RPC вызовами, состоянием игроков, синхронизацией очков и генерацией кодов комнат. Поддерживает проброс UPnP. |
| 📡 `LANDiscovery.gd` | Broadcaster и Listener UDP-пакетов. Отвечает за автоматическое обнаружение серверов преподавателей в школьных локальных сетях. Облегчает подключение для детей. |
| 🧠 `GeminiAPI.gd` | Обёртка LLM интерфейса (взаимодействие с API Groq и Google Gemini). Занимается асинхронными HTTP запросами к нейросетям, генерацией вопросов, валидацией ответов и формированием индивидуального "Плана изучения". Оснащена системой Auto-Fallback при 429 ошибках (Rate Limit). |
| 🎬 `GameManager.gd` | State Machine игры (Лобби -> Подготовка -> Вопрос -> Оценка -> Лидерборд). Синхронизирует фазы раундов между всеми клиентами. Хост управляет временем, а `GameManager` следит, чтобы все перешли на один экран. |
| 📊 `StudentHistory.gd` | Модуль локальной статистики и аналитики (сохраняется в `user://`). Считает слабые зоны, средние баллы и генерирует красивые форматированные документы `.txt` для выгрузки (Экспорт аналитики группы и Плана обучения). |
| 🖥️ `UI_Manager.gd` | Концентратор всех `Control` нодов. Отвечает за анимацию шторки экранов, окна диалогов, красивые цвета, панели ответов, настройки и всплывающие уведомления. |
| 🏗️ `PillarGenerator.gd` / `PlayerController.gd` | Визуализация успехов студентов в виде растущих колонн в 3D мире. Использует физику и `Tween` интерполяции для сглаженного "взлета" аватаров. |

## 🚀 Установка и Запуск (Для Разработчиков)

1. Скачайте и установите **Godot Engine** версии 4.2+ с [официального сайта](https://godotengine.org/download/).
2. Скачайте этот репозиторий: `git clone https://github.com/ваше-имя/neurotower.git`
3. Откройте приложение Godot, нажмите **"Импорт" (Import)** и выберите файл `project.godot` в корневой папке проекта.
4. В скрипте `GeminiAPI.gd` вставьте ваши API-ключи:
   - Создайте аккаунт в [Groq Cloud](https://console.groq.com/keys) для ключа `API_KEY`.
   - Создайте проект в [Google AI Studio](https://aistudio.google.com/app/apikey) для запасного ключа `GOOGLE_KEY`.
5. Нажмите кнопку **"Запуск" (F5)**!
